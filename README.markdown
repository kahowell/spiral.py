Spiral
======
spiral.py is a python3 program that prints integer spirals. The spirals are
counter-clockwise spirals that stop at a given number

Example:

    ./spiral.py 6

    6
    5 0 1
    4 3 2

Usage
-----

    ./spiral.py <number>
